Selecting Files

Select the files you desired to have the lines of text counted by using the "Get Files" button.  The standard Windows file open dialog will appear.  Browse for and select all the files you want counted and click the open button.

If one file was selected, that file will be displayed in under "Location of Files:" in the sunken static box.  This box will contain the directory of the selected files if more than one file was selected.

In addition, the message "Files ready to count." will be displayed in the lower left hand corner.

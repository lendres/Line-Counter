Counting Lines

After selecting the desired files click the "Count" button.  That's it!!

Each file will be examined and the lines counted.  The sunken static boxes will be updated with the number of files and total number of lines after each file is finished.  In addition, the message "Counting..." will be displayed in the lower left hand corner while the program is working.

When all the files are done the message "Finished." will be displayed in the lower left hand corner.
